# Vim Shortcut Commands

| key | Description |
| ---- | ---- |
| 0 | 行頭へ移動(インデント無視) |
| ^ | 行頭に移動 |
| $ | 行末へ移動 |
| j | 下に移動 |
| k | 上に移動 |
| h | 左に移動 |
| l | 右に移動 |
| 100j | 100行下に移動 |
| 100k | 100行上に移動 |
| dd | 一行カット |
| yy | 一行コピー |
| p | 貼り付け |
| u | Undo |
| [Ctrl] + r | redo |
| w | 次の単語の先頭に移動 |
| e | 今の単語の末尾に移動 |
| x | 一文字削除 |
| a | 次の位置からInsert |
| i | 現在の一からInsert |
| o | 次の行からInsert |
| : | Command Modeに切り替え |
| g | 最初に移動 |
| G | 最後に移動 |
| gv | 前回の選択位置を選択 |
